CSS(Cascading Style Sheets)
	HTML 문서를 시각적으로 꾸며주는 역할을 하는 언어
	요소가 구체적으로 어떤 스타일로 표현되는지를 정의하는 규격
	HTML 내부적으로 요소의 디자인을 할 수 있지만,
	가독성이 떨어지고 유지보수가 좋지 않으므로 해결하기 위해 CSS를 사용
	(구조, 내용과 스타일을 분리하여 분업을 한다)

CSS 문법
	선택자 {
		속성명 : 속성값;
		속성명 : 속성값;
		...
	}

	ex) p { color: blue; text-align:center; }
	    --    ----------------------------------
	 선택자		선언부

	p태그를 선택하여 가운데 정렬하고 파란색 글씨로 설정

	HTML 문서 중 디자인을 적용시킬 대그 안에 style 속성에 작성
	HTML 문서 <head> 태그 안에 <style> 태그 안에 css 문법을 작성
	.css 파일을 만들어서 그 안에 css 문법을 작성

CSS를 적용하는 방법
	1. 인라인 스타일
		HTML 요소 내부에 style 속성을 사용해서 적용하는 방법
		해당 요소에만 스타일을 적용(선택자가 필요없다)
	2. 내부 스타일 시트
		HTML 문서의 <head> 태그 안에 <style> 태그를 사용해서 적용하는 방법
		해당 문서에서만 스타일을 적용(선택자가 꼭 필요함)
	3. 외부 스타일 시트
		웹 사이트 전체의 스타일을 하나의 파일에서 변경, 관리
		.css 파일을 만들어 CSS 문법을 작성하고 디자인을 적용시킬 곳마다
		<head> 태그 안에 <link> 태그를 사용해서 연결, 적용하는 방법

		<link href="css파일경로" rel="stylesheet">
		rel 속성은 현재 파일과 링크된 파일 사이의 연관관계를 명시

CSS 선택자
	1. 전체 선택자
		스타일을 모든 요소에 적용
		디테일한 선택자로 적용시킨 스타일은 전체 선택자나 body에 적용한 
		스타일보다 더 우선시 됨
		
		* {
			css 문법
		}

	2. 태그선택자, 요소 선택자
		특정 태그가 쓰인 모든 요소에 스타일을 적용

		태그이름 {
			css 문법
		}

	3. id 선택자
		웹 문서 안의 특정한 요소, 부분에 스타일을 적용
		html 요소의 id 속성값을 이용
		# 기호를 이용

		#아이디명 {
			css 문법
		}

	4. class 선택자
		특정 집단의 요소에 한번에 스타일을 적용할 떄 사용
		html 요소의 class 속성값을 이용	
		. 기호를 이용

		.클래스명{
			css 문법
		}
		
	5. 그룹 선택자
		여러개의 선택자를 나열하고 ,로 구분해 스타일을 적용

		선택자1, 선택자2{
			css 문법
		}

	6. 자손 선택자
		조상요소 하위에 있는 자손을 선택해 스타일을 적용
		자손은 자식을 포함

		조상요소선택자 자손요소선택자{
			css 문법
		}

		p.txt : p태그들 중에 txt클래스를 가진 요소 선택
		p .txt : p태그들 안에 있는 txt클래스를 가진 요소들 선택
	
	7. 자식 선택자
		부모 요소 하위에 있는 자식을 선택해 스타일을 적용
		
		부모선택자 > 자식선택자 {
			css 문법
		}

	8. 인접 형제 선택자	
		형제 관계에서 연속적으로 존재하는 요소를 선택할 때 사용
		
		형선택자 + 동생선택자{
			css 문법
		}	

		a + li : a 태그를 바로 위의 형으로 가지고 있는 li태그 선택
